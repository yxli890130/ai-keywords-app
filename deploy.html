部署到 GitHub Pages

获取免费的 HTTPS 链接，分享给家人使用

1 创建 GitHub 仓库

📍 打开 https://github.com/new

📍 填写仓库信息：

  • Repository name: 输入 ai-keywords-app（或其他名称）
  • Public: 选择 公开（Private 无法使用免费 Pages）
  • 不要勾选 "Add a README file"

📍 点击 "Create repository" 按钮

⬇️
2 上传项目文件

📍 在新创建的仓库页面，点击 "uploading an existing file"

📍 将以下文件拖入上传区域：

  • index.html
  • manifest.json
  • service-worker.js
  • browserconfig.xml
  • README.md
  • install.html
💡 注意：暂时不上传 icons 文件夹（图标是可选的）

📍 滚动到底部，填写提交信息，点击 "Commit changes"

⬇️
3 启用 GitHub Pages

📍 进入仓库的 Settings（设置）页面

📍 在左侧菜单找到 "Pages"

📍 配置如下：

  • Source: 选择 Deploy from a branch
  • Branch: 选择 main 或 master
  • Folder: 选择 / (root)

📍 点击 "Save" 按钮

⬇️
4 获取 HTTPS 链接

📍 等待 1-2 分钟，刷新 Pages 页面

📍 你会看到这样的链接：

https://你的用户名.github.io/ai-keywords-app/

📝 复制这个链接，通过微信发送给家人！

✅ 部署成功！

你的应用链接：

https://你的用户名.github.io/ai-keywords-app/

家人使用方法：

  1. 在微信中点击链接
  2. 选择 "在 Safari 中打开"
  3. 点击分享按钮 → "添加到主屏幕"
  4. 点击 "添加"

🎉 完成！家人现在可以像使用普通 App 一样使用了！

🎨 可选：添加应用图标

如需添加图标：

  1. 在浏览器中打开 generate-icons.html
  2. 等待所有图标自动下载
  3. 在 GitHub 上创建 icons 文件夹
  4. 上传下载的图标文件
🌐 其他免费托管平台

除了 GitHub Pages，还可以使用：

  • Vercel - vercel.com （拖拽部署）
  • Netlify - netlify.com （拖拽部署）
  • Cloudflare Pages - pages.cloudflare.com
