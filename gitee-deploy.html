🇨🇳 部署到 Gitee Pages

国内直连访问，无需 VPN

💡 为什么选择 Gitee？
  • ✅ 国内访问速度极快
  • ✅ 无需 VPN 即可访问
  • ✅ 完全免费使用
  • ✅ 支持自定义域名
1 创建 Gitee 仓库

📍 打开 Gitee 创建页面：

打开创建页面 →

📍 填写仓库信息：

  • 仓库名称: ai-keywords-app
  • 仓库介绍: AI绘画关键词宝库
  • 是否公开: 选择 公开
  • 初始化仓库: 不要勾选任何选项

📍 点击 "创建" 按钮

⬇️
2 推送代码到 Gitee

📍 在项目目录运行以下命令：

cd /d/claudecode/【问今AI】关键词宝藏库/AI关键词宝库PWA
git remote add gitee https://gitee.com/你的用户名/ai-keywords-app.git
git push -u gitee main

或直接在 Gitee 网页上传文件

⬇️
3 启用 Gitee Pages

📍 进入仓库页面，点击 "服务" → "Gitee Pages"

📍 配置如下：

  • 部署分支: main
  • 部署目录: 根目录

📍 点击 "启动" 或 "更新" 按钮

⬇️
4 获取国内访问链接

📍 等待部署完成后，你的国内访问链接为：

https://你的用户名.gitee.io/ai-keywords-app

✅ 完成！

现在家人在国内可以直接访问！
