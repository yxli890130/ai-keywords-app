🔑 创建 GitHub Personal Access Token

快速创建Token，用于自动部署应用

1 打开 GitHub Token 创建页面

📍 点击下方按钮或访问：

https://github.com/settings/tokens/new
⬇️
2 配置 Token 设置

📍 在页面填写以下信息：

  • Note（备注）: 输入 AI Keywords App
  • Expiration（过期时间）: 选择 No expiration 或 90 days
  • Scopes（权限）: 勾选 repo（完整仓库权限）
💡 提示: 只需要勾选 repo 这一栏，其他不需要勾选
⬇️
3 生成并复制 Token

📍 点击页面底部的 "Generate token" 按钮

📍 重要: 复制生成的 Token（只会显示一次！）

⚠️ 注意: Token 形如 ghp_xxxxxxxxxxxx，请立即复制保存
⬇️

📝 粘贴你的 Token

粘贴后点击"继续"，将自动创建仓库并部署应用：

❤️ Token 只用于此次部署，不会被保存
