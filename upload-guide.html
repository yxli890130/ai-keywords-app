📤 上传文件到 GitHub

1 打开仓库页面

📍 点击下方按钮打开你的GitHub仓库：

打开仓库 →
⬇️
2 上传文件

📍 在仓库页面，点击 "uploading an existing file"

📁 拖入以下文件：

index.html manifest.json service-worker.js browserconfig.xml README.md install.html deploy.html QUICK_START.md token-guide.html generate-icons.html icon-generator.html

📍 滚动到底部，填写提交信息：

Commit message: Initial commit

📍 点击 "Commit changes" 按钮

⬇️
3 启用 GitHub Pages

📍 进入仓库的 Settings（设置）页面

📍 在左侧菜单找到 "Pages"

📍 配置如下：

  • Source: Deploy from a branch
  • Branch: main
  • Folder: / (root)

📍 点击 "Save" 按钮

✅ 完成！
